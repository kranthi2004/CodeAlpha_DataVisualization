# CodeAlpha Task 3 - Data Visualization

## Project Overview
This project focuses on transforming raw book dataset into meaningful visualizations using Python libraries such as Matplotlib and Seaborn. The visualizations help identify trends, patterns, and business insights.

## Dataset
- Dataset: Books Dataset
- Records: 1000 Books

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Visualizations Created
- Price Distribution
- Rating Distribution
- Top 10 Most Expensive Books
- Price vs Rating Box Plot
- Correlation Heatmap
- Book Availability Pie Chart
- Dashboard
- Business Insights

## Business Insights
- Most books are priced between £10 and £60.
- Most ratings are between 2 and 4 stars.
- Price has very little correlation with rating.
- Nearly all books are available in stock.
- High-priced books are not always highly rated.
